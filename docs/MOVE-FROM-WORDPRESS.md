# How to Make pastrypleasures.co.ke Rank First for "Pastry Pleasures"

The old site **pastrypleasures.wordpress.com** still appears first on Google for "pastry pleasures" because it has been online longer. To get **pastrypleasures.co.ke** to rank first, do the following.

---

## 1. Redirect the Old WordPress Site (Most Important)

When the old site **permanently redirects (301)** to the new one, Google treats the new site as the main address and can transfer trust over time.

### Steps on WordPress.com

1. **Log in** to [WordPress.com](https://wordpress.com) and open the dashboard for **pastrypleasures.wordpress.com**.

2. **Buy a Site Redirect** (paid add-on):
   - Go to **Upgrades → Domains** (or **Hosting → Domains** in WP Admin).
   - Find your site address (e.g. **pastrypleasures.wordpress.com**).
   - Click the **…** next to it and look for **Site Redirect** or **Redirect this site**.
   - Or open: [wordpress.com/domains/add/site-redirect/](https://wordpress.com/domains/add/site-redirect/)
   - Purchase the **Site Redirect** and set the redirect URL to:
   - **https://pastrypleasures.co.ke** (or **http://pastrypleasures.co.ke** if the form only allows http; your host will handle https).

3. **Set the redirect as primary**  
   In **Upgrades → Domains**, set the Site Redirect address as the **primary site address** so all traffic (including from Google) goes to pastrypleasures.co.ke.

4. **Confirm**  
   Visit **pastrypleasures.wordpress.com** in a browser; it should redirect to **pastrypleasures.co.ke**.

After this, Google will see the old URL as permanently moved to the new one. Rankings can take a few weeks to shift.

---

## 2. Tell Google You Moved (Google Search Console)

This helps Google consolidate the old and new site in search.

1. **Add both properties** in [Google Search Console](https://search.google.com/search-console):
   - Add **https://pastrypleasures.wordpress.com** (if you still have access).
   - Add **https://pastrypleasures.co.ke** (your current site).

2. **Submit the new sitemap** for pastrypleasures.co.ke:
   - In Search Console, select the property **pastrypleasures.co.ke**.
   - Go to **Sitemaps**.
   - Submit: `https://pastrypleasures.co.ke/sitemap.xml`.

3. **Use “Change of Address”** (if available):
   - In Search Console, open **Settings** for the **pastrypleasures.co.ke** property.
   - If you see **Change of address**, use it to indicate you moved from **pastrypleasures.wordpress.com** to **pastrypleasures.co.ke**.
   - This is only shown when both properties are verified.

4. **Request indexing** for the new homepage:
   - Use **URL Inspection** for `https://pastrypleasures.co.ke/`.
   - Click **Request indexing** so Google recrawls with the new content and redirect in place.

---

## 3. What We Did on pastrypleasures.co.ke

On the new site we already:

- Set a clear **title** and **meta description** (e.g. “Pastry Pleasures – Bakery, Cakes & Recipes | Nairobi, Kenya”).
- Added **structured data** (WebSite + Organization) so Google knows the official business and URL.
- Added **“Pastry Pleasures official website – pastrypleasures.co.ke”** in the footer so the brand and domain are clearly linked.
- Set a **canonical URL** and **Open Graph** tags.
- Updated the **sitemap** and key page descriptions.

Together with the **301 redirect** and **Search Console** steps above, this gives Google a clear signal that **pastrypleasures.co.ke** is the main, official site for “Pastry Pleasures.”

---

## Summary

| Action | Where | Effect |
|--------|--------|--------|
| 301 redirect old → new | WordPress.com (Site Redirect) | Tells Google the new site is the main one |
| Add both sites + sitemap | Google Search Console | Helps discovery and consolidation |
| Change of address | Search Console (if shown) | Reinforces the move |
| Request indexing | Search Console | Speeds up recrawl of the new site |

After the redirect is active and Search Console is set up, give it **2–6 weeks** for Google to re-crawl and update rankings. The redirect is the single most important step.
